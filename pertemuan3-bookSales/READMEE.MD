# Tugas SIB Mandiri - BookSales App

Repository ini berisi pengerjaan tugas pengembangan aplikasi web menggunakan React JS dengan fokus pada pengelolaan data dinamis dan state management.

## ✅ Progres Pengerjaan
Sesuai dengan instruksi tugas, berikut adalah poin-poin yang telah diselesaikan:

1. **Struktur Folder:** Membuat folder `Utils` di dalam folder `src` untuk manajemen data yang rapi.
2. **Database Lokal:** Membuat file `src/Utils/books.js` yang berisi array of objects.
3. **Data Koleksi:** Menyediakan total **9 data buku** yang berbeda (termasuk referensi buku asli seperti Teori Bahasa Automata dan Sejarah WW1).
4. **Render Data:** Menampilkan data buku di halaman aplikasi menggunakan metode `.map()`.
5. **Fitur State (Nilai Tambah):** Menambahkan tombol (button) untuk menambah data buku baru secara dinamis menggunakan React Hooks (`useState`).

## 🛠️ Teknologi
- **React JS** (Vite)
- **Bootstrap 5** (Styling UI)
- **JavaScript ES6** (Array Methods & Spread Operator)

## 📂 Struktur File Penting
- `src/Utils/books.js`: Tempat penyimpanan 9 data buku.
- `src/Pages/Book.jsx`: Komponen untuk merender list buku dan logika tambah buku.
- `public/`: Berisi aset gambar lokal agar tampilan stabil.
