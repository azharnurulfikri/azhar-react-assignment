# Tugas Pertemuan 1: BookSales Landing Page

Tugas ini adalah bagian dari program SIB Mandiri NF Academy yang berfokus pada implementasi dasar **React.js** dan **Bootstrap**.

## ✨ Fitur Utama
- **Navbar:** Navigasi responsif dengan logo kustom (BookSales).
- **Main Hero:** Menampilkan buku unggulan (Atomic Habits) dengan tombol CTA.
- **Album Grid (3x3):** Menampilkan koleksi buku menggunakan array mapping di React.
- **Team Section:** Menampilkan profil tim kreatif (Lead Developer: Azhar).
- **Contact Form:** Formulir kontak sederhana menggunakan Bootstrap Form.
- **Footer:** Navigasi bawah dan informasi copyright.

## 🛠️ Tech Stack yang Digunakan
- **React.js** (Functional Components)
- **Vite** (Build tool)
- **Bootstrap 5** (Styling & Layouting)
- **React Icons / FontAwesome** (Icons)

## 📝 Catatan Perkembangan
- Berhasil mengimplementasikan sistem *scroll-to-id* (smooth scrolling) untuk navigasi antar bagian halaman.
- Menggunakan looping array untuk merender kartu album agar kode lebih bersih (tidak redundan).
- Frame album dimodifikasi dengan `rounded-4` dan `shadow-sm` untuk tampilan modern.
